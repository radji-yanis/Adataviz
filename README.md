# L'Agenda Parisien 🗓️

Petite appli JavaScript qui affiche les événements culturels à Paris, à partir de l'Open Data de la Ville de Paris.

## Fonctionnalités

- Recherche d'événements
- Pagination
- Filtre par prix
- Thème clair / sombre

## Utilisation

Ouvrir `index.html` dans un navigateur.

## Tests

```bash
npx vitest
```
